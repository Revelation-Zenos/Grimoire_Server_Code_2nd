# Grimoire_Server
このリポジトリは、レヴェルシオン・ゼノス(旧名 レミリア)氏が運営する、
グリモア鯖の公式リポジトリです。
本番もといもととなるファイルはグリモア機にあります。
コードスペースにあるファイルはグリモア機から引っ張ってきたが少し古めの模造品ファイル。
(普通に古いし、本体はグリモア機にある。)

mainブランチには追加してほしいもの編集を加えてもらいたいものをアップロード、追加してもらって全然構わないです。
(ただ、関係のないものをアップロードしていたのを見つけた場合、削除します。)

新しく来た編集者に向けてのレヴェルシオン・ゼノス氏からのコメント
コードスペースに、グリモア鯖に関するファイルが入ってる、
編集してもらって構わないが、編集する前にルールをよく読んでから編集してほしい。
そうじゃないとこちらが困る。

グリモア機及びアイギアス機はレヴェルシオン・ゼノス氏が所有しているゲーム用に改造したワークステーションPCであり、
サーバー及び、所有者はレヴェルシオン・ゼノス氏である事を忘れないで下さい。
