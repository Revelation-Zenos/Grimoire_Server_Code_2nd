# Grimoire_Server
このリポジトリは、レヴェルシオン・ゼノス(旧名 レミリア)氏が運営する、
グリモア鯖の公式オープンリポジトリです。
本番環境用のファイルと元となるファイルはグリモア機にあります。
コードスペースにあるファイルはグリモア機から引っ張ってきた模造品ファイル。
(ファイルの本体はグリモア機にある。)

mainブランチには追加してほしいもの編集を加えてもらいたいものをアップロード、追加してもらって全然構わないです。
(ルール6に則って関係のないものをアップロードしていたのを見つけた場合、削除します。)

新しく来た編集者に向けてのレヴェルシオン・ゼノス氏からのコメント
コードスペースに、グリモア鯖に関するファイルが入ってる、
編集してもらって構わないが、編集する前にルールをよく読んでから編集してほしい。
そうじゃないとこちらが困る。

グリモア機及びアイギアス機はレヴェルシオン・ゼノス氏が所有しているゲーム兼配信兼サブPC用に改造したグリモア鯖のベース機は
Dell Precision T7600(Dual Xeon E5-2667)ワークステーションPCであり、
サーバー及び、所有者はレヴェルシオン・ゼノス氏である事を忘れないで下さい。
